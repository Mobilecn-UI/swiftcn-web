## Why Swiftcn UI?

- **Customizable**: Every component is designed to be easily customized to fit your application's specific needs.
- **Easy to Use**: Just include the component files in your project, and you're good to go! No complex setup required.

- **Beautifully Designed**: With a focus on aesthetics, each component is carefully crafted to provide a visually appealing experience.

- **Built for SwiftUI**: Native support for SwiftUI ensures that you can make the most out of the latest Apple technologies.

## Installation

[Swiftcn UI GitHub Repository](https://github.com/Swiftcn-UI/swiftcn-playground/tree/main/Swiftcn%20Playground.swiftpm/Components)

Add the components folder and easily import the components, start building beautiful SwiftUI apps.
